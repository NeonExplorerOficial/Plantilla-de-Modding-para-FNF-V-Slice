# scripts

Esta carpeta contiene funciones y clases programadas que utiliza el juego.


¡Dato curioso! Aunque las funciones programadas requieren que el script de destino se encuentre en una ubicación específica, la clase programada puede estar en cualquier lugar siempre que amplíe la clase de destino adecuada. La estructura de directorios para las clases programadas tiene principalmente fines organizativos.
